Q2: CSS

The Quiz

Use the instructions below to style the quiz using CSS.

  1. Create a CSS file called style.css and link to it.
  2. Make all of the headings a sans serif font.
  3. Make the name of the course all uppercase.
  4. Give the name of the course 1px letter spacing.
  5. Make the name of the course have a size of .8em.
  6. Give Q2: CSS a bottom border that is 3px, solid, and black.
  7. Make the instructions text italicized.
  8. Give each of the questions a bottom margin of 1em.
  9. Remove the left margin from the questions.
  10. Make the footer background blue.
  11. Make the footer text white.
  12. Make the footer links sans serif.
  13. Make the footer links bold.
  14. Make the footer links yellow.
  15. Give the footer content a padding all around of 1em.
  16. Write down five questions you still have about CSS.

Turning In Your Quiz
